# подсказка по командной строке

## как откыть терминал
1. win+r
2. прописать cd

## как сменить директорию
### windows
```sh
cd C:\folder_name
```
### macoc/linux
```sh
cd /folder_name
```
## листинг директории
### windows
```sh
dir
```
### macos/linux
```sh
ls
```
## удалить файл
### windows
``` sh
del name_fail
```
### macos/linux
``` sh
rm name_fail
```
